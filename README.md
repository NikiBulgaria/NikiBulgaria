#### 1. ESP32-S3 Show Controller (Python/PyQt6) - *Latest*
A complete Python port of a custom Unity app I built to control a live physical show. 
*   **The App:** Built with Python and PyQt6, featuring a fullscreen kiosk show mode, a physics-based prize wheel, and a hidden settings dashboard.

#### 2. Smart Farm IoT Ecosystem (2025)
My 12th-grade graduation project. A fully automated, self-sustaining greenhouse ecosystem built from scratch.
*   **Custom Hardware:** Powered by an ESP32. I engineered the power delivery with an auto-switching 12V/220V circuit (using a repaired LED driver and a 12V backup battery). It reads from DHT11, soil moisture, and voltage sensors to drive a 4-channel relay controlling a water pump, cooling fan, and a custom heater.
*   **Tri-Mode Logic:** The system is virtually unkillable. 
    *   *Online:* UDP Wi-Fi link talking to my custom app. 
    *   *Bluetooth:* Serial fallback for local configuration.
    *   *Autonomous (Offline):* If the network drops, it reads the last known instructions from its EEPROM memory to keep the plants alive automatically.
*   **The UI:** I designed a user-friendly Unity app for remote monitoring (water levels, battery percentage, live sensor graphs) and configuring the farm's thresholds.
